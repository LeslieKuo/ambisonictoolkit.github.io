The ATK library for [SuperCollider3](http://supercollider.sourceforge.net/) is distributed via the [sc3-plugins](http://sc3-plugins.sourceforge.net/) project.

On Mac OS X, the plugin download can be placed into the ~/Library/Application Support/SuperCollider/Extensions folder. If building from source for OS X or Linux, follow the sc3-plugins build and install directions.

Additionally, the SuperCollider3 version of the ATK has a number of dependencies. Please install the following

Install the MathLib(external link) Quark via the instructions found here(external link).
Download and install [ATK Kernels](/download/kernels)
Download and install [ATK Sound File Example Recordins](/download/recordings)

NOTE: The ATK requires SuperCollider3 version 3.5. See [here](http://sourceforge.net/projects/supercollider/files/) for the latest up-to-date distribution and [here](http://supercollider.sourceforge.net/developers/) for instructions on retrieving source code.

## Source code

You can build the SuperCollider3 version of the ATK from the [sc3-plugins](https://sourceforge.net/projects/sc3-plugins/) source-code. The ATK is free software, and is published under the [GPLv3](http://www.gnu.org/copyleft/gpl.html) free software license.
